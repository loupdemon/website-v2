Les addons

Cette page reprend tous les addons présents dans Scratch Addons.

Cette page est incomplète, et nous avons besoin de votre aide pour ajouter des images. En savoir plus ici.

Sélecteur de couleurs 2D
2d-color-picker

Remplace les curseurs de saturation et de luminosité par un sélecteur de couleurs 2D. Maintenez Maj. enfoncé tout en déplaçant le curseur pour changer les valeurs sur un seul axe.

Mode lecture de projet en 60FPS ⭐
60fps

Alt + Clic sur le drapeau vert pour activer le mode 60FPS.

Corriger les majuscules dans les paramètres du compte 🥚
account-settings-capitalize

Change "Paramètres du compte" dans le menu du compte en "Paramètres du Compte".

Sélecteur de miniature ⭐
animated-thumb

Ajoute un boutton à l'éditeur sui permet de changer la miniature du projet avec n'importe quelle image (y compris les gifs).

Meilleurs emojis
better-emojis

Remplace les émojis chat dans les commentaires Scratch par des versions améliorées en vectoriel.

Bannière de page de profil
better-featured-project

Transforme le projet présenté sur n'importe quel profil d'utilisateur en une jolie bannière.

Chargement d'images en HD
better-img-uploads

Ajoute un nouveau bouton au-dessus du bouton "upload le costume" qui convertit automatiquement les images bitmap téléchargées en images SVG (vecteur) pour éviter de perdre en qualité.

Meilleures citations sur le forum
better-quoter

Sélectionnez du texte dans un post de forum et cliquez sur le bouton "citation" dans le coin en bas à droite pour citer uniquement le texte sélectionné.

Copie des images bitmap
bitmap-copy

Vous permet de copier une image bitmap de l'éditeur d'images, pour pouvoir la coller ensuite dans d'autres sites ou logiciels.

Prendre un seul bloc avec la touche Ctrl
block-cherry-picking

Ajoutes la possibilité de déplacé les blocs un par un avec la touche Ctrl (plutôt que le script le suivant)

Compteur de Blocs
block-count

Affiche le nombre total de blocs d'un projet dans la barre de menus de l'éditeur. Faisait auparavant partie de "nombre de sprites et de scripts".

Dupliquer le script avec la touche Alt
block-duplicate

Dupliquez rapidement un script en le déplaçant tout en maintenant la touche Alt enfoncée. Maintenez la touche Ctrl enfoncée pour ne dupliquer qu'un seul bloc au lieu de toute la pile attachée en dessous.

Icônes des catégories de la palette des blocs
block-palette-icons

Ajoute des icônes dans les cercles colorés qui identifient les catégories de blocs

Échanger de bloc ⭐
block-switching

Faites un clic droit sur un bloc pour le remplacer par un autre bloc associé.

Enregistrer les blocs en tant qu'image
blocks2image

Faites un clic droit sur la zone de code de l'éditeur pour exporter les blocs sous formes d'images SVG / PNG.

Blocs chat
cat-blocks

Ramène les blocs chapeaux avec une tête de chat, venant du poisson d'avril 2020.

Compteur de clones ⭐
clones

Ajoute un compteur dans l'éditeur pour voir le compte total de clones.

Jeux en ligne ⭐
cloud-games

Disponible lorsque l'icone Scratch Addons est cliqué. Vous montre combien d'utilisateurs sont actuellement en train de jouer sur des projets mulitjoueurs, et donne leurs pseudonymes.

Palette de couleurs hexa ⭐
color-picker

Ajoute une entrée pour code hexa aux palettes de couleurs.

Menu catégorie à deux colonnes
columns

Divise le menu des catégories de blocs en deux colonnes et le déplace vers le haut de la palette de blocs, comme dans Scratch 2.0

Sauts de ligne dans les commentaires
comments-linebreaks

Rend les sauts de ligne (en appuyant sur Entrée) dans les commentaires sur le site Web visibles au lieu de les convertir en espaces.

Messages compacts
compact-messages

Rendre la page des messages de Scratch plus compacte et plus facile à lire.

Confirmations ⭐
confirm-actions

Vous demande si vous êtes sûr lorsque vous effectuez des actions sous votre compte Scratch, telles que le partage de projets.

Bouton copier le lien d'un commentaire ⭐
copy-message-link

Ajoute un bouton "Copier le lien" aux commentaires sur Scratch, à côté du bouton "Signaler".

Ctrl + Clic pour exécuter les scripts
ctrl-click-run-scripts

Exécuter les scripts quand vous cliquez dessus uniquement si la touche contrôle est pressée

Ctrl + Enter pour envoyer
ctrl-enter-post

Publie automatiquement le commentaire ou la publication du forum avec Ctrl+Entrée.

Lien des conservateurs sur la page d'accueil
curator-link

Fait du nom d'utilisateur du conservateur de la page d'accueil un lien cliquable vers son profil.

Forme de bloc personnalisable
custom-block-shape

Ajustez les marges, le rayon des coins et la hauteur de l'entaille des blocs Scratch.

Style de texte pour les blocs personnalisables
custom-block-text

Change l'épaisseur du texte des blocs et ajoute une ombre optionnelle.

Zoom de la zone de code personnalisable
custom-zoom

Ajuste le minimum, le maximum, la vitesse et la taille de base du zoom dans l'éditeur de code du projet, et cache automatiquement les contrôles.

Bordure d'image de profil personnalisable
customize-avatar-border

Vous permet de masquer ou de modifier le contour des images de profil dans les commentaires de projet et de studio.

Pluie de dangos sur les profils (Poisson d'avril) 🥚
dango-rain

Les utilisateurs ayant ajouté le mot "dango" dans leur section "À Propos de moi" auront une pluie d'émojis de dangos sur les côtés de leur profil.

Mode sombre du site Web et couleurs personnalisables
dark-www

Personnalisez les couleurs du site Web de Scratch. Plusieurs thèmes sombres de différents personnes disponibles. Si vous n'avez pas besoin du mode sombre mais que vous souhaitez modifier les couleurs par défaut, vous pouvez sélectionner le préréglage "Couleurs par défaut de Scratch" et le modifier.

Améliorations pour la catégorie des blocs de donnée ⭐
data-category-tweaks-v2

Apporte des améliorations pour la catégorie des blocs de donnée ("Variables") dans l'éditeur.

Debugger
debugger

Ajoute une nouvelle fenêtre "debugger" à l'éditeur. Permet de se connecter à l'onglet "Console" de la fenêtre du débogueur à l'aide des blocs "log", "warn" et "error". Le bloc "point d'arrêt" mettra le projet en pause lors de son exécution. Tous les threads en cours d'exécution peuvent être visualisés dans l'onglet "Threads" du débugger, et lorsqu'il est en pause, le bouton "Step" peut être utilisé pour exécuter le bloc suivant. Un graphique des FPS et du nombre de clones peut être visualisé dans l'onglet "Performance".

Projet par défaut personnalisable
default-project

Remplacez le projet par défaut de Scratch Cat par n'importe quel projet existant.

Désactiver la sauvegarde automatique
disable-auto-save

Désactive la sauvegarde automatique du projet durant sa modification.

Ne pas déplacer les éléments collés
disable-paste-offset

Collez les éléments copiés à leur position d'origine au lieu de les déplacer légèrement dans l'éditeur de costumes.

Désactiver l'oscillation des sprites dans le volet des sprites
disable-sprite-wobble

Autorise le déplacement des scripts, des costumes et des sons vers différents sprites sans que les sprites ne se déplacent dans le volet des sprites.

Sprites non glissables dans l'éditeur
disable-stage-drag-select

Enlève la possibilité de déplacer les lutins dans la scène, sauf s'ils sont définis explicitement comme glissables.

Barre de navigation personnalisable
discuss-button

Choisis une liste de liens à afficher dans la barre de navigation.

Glisser-déposer de fichier ⭐
drag-drop

Vous permet de glisser des images ou des sons depuis votre explorateur de fichiers directement dans la listes des sprites, des sons et des costumes. Vous pouvez également faire glisser des fichiers texte dans les listes ou dans la boîte de dialogue "demander et attendre".

Bouton de l'effet écho
echo-effect

Remet le bouton Echo dans l'éditeur de sons.

Menus déroulants colorés
editor-colored-context-menus

Menus contextuels colorés quuan on fait un clic droit sur un bloc

Prévisualisation des commentaires dans l'éditeur
editor-comment-previews

Vous permet de prévisualiser le contenu des commentaires en survolant les commentaires et blocs réduits. Vous pouvez l'utiliser pour afficher les commentaires hors écran, identifier un bloc de boucle à partir du bas par son aperçu, insérer de nombreux commentaires longs dans un petit espace, etc.

Mode sombre de l'éditeur et couleurs personnalisables
editor-dark-mode

Personnaliser les couleurs utilisées dans l'éditeur de projets. Plusieurs thèmes sombres créés par différents auteurs sont disponibles. Si vous ne voulez pas utiliser le mode sombre mais que vous voulez changer les couleurs utilisées par défaut, vous pouvez sélectionner le préréglage " Couleurs par défaut de Scratch " et le modifier.

Outils pour développeurs ⭐
editor-devtools

Ajoute une une barre de recherche et de nouvelles options de menus à l'éditeur : copier/coller des blocs, un meilleur nettoyage, aller à la définition d'un bloc personnalisé, et plus !

Prise en charge de touches supplémentaire
editor-extra-keys

Ajoute plus de touches aux blocs "touche () enfoncée ?" et "quand la touche () est pressée", les listes déroulantes de blocs, telles que entrée, point, virgule, etc.

Messages dans l'éditeur ⭐
editor-messages

Affiche votre nombre de messages dans l'éditeur

Menus déroulants recherchables ⭐
editor-searchable-dropdowns

Vous permet de rechercher des listes déroulantes de blocs.

Effets sonores dans l'éditeur
editor-sounds

Produit un son lorsque vous connectez ou supprimez des blocs.

Afficher la scène sur le côté gauche
editor-stage-left

Déplace la scène sur le côté gauche de l'éditeur.

Bordure du bloc executé
editor-stepping

Ajoute un contour bleu sur les blocs qui sont actuellement exécutés dans un projet.

Couleurs des blocs personnalisable
editor-theme3

Personnalisez la couleur des blocs pour chaque catégorie de bloc dans l'éditeur.

Bouton Annuler le partage dans l'éditeur
editor-unshare-button

Remplace le bouton "Partagé" à côté de la zone de titre du projet par un bouton pour annuler le partage du projet.

Sélecteur d'emoji pour les commentaires
emoji-picker

Ajoute un bouton pour mettre des emojis dans les commentaires d'un projet.

Montrer le nombre exact ⭐
exact-count

Montre le compte exact pour les informations d'utilisateurs et de studios. Utilise ScratchDB pour les informations concernant les forums.

Barre de recherche expansible
expanding-search-bar

Rendre la barre de recherche plus grande lorsqu'elle est active.

Projets non partagés à l'affiche
feature-unshared

Vous permet de choisir un projet non partagé au moment de changer le projet à l'affiche sur votre page de profil.

Meilleurs commentaires dans l'éditeur ⭐
fix-editor-comments

Apporte de nombreuses modifications aux commentaires dans l'éditeur Scratch : corrige un bug où les commentaires attachés aux blocs n'enregistrent pas correctement leurs positions après avoir été glissés, empêche les commentaires de bloc d'apparaître hors écran dans les scripts volumineux, maintient les connexions de commentaire de bloc droites, et plus encore.

Ne pas exécuter automatiquement les blocs dupliqués ⭐
fix-pasted-scripts

Corrige un bug dans l'éditeur Scratch, où dupliquer un bloc et l'attacher à un script entraîne l'exécution du script.

Corrige les uploads SVG
fix-uploaded-svgs

Corrige un bug dans l'éditeur Scratch qui rend les fichiers SVG uploadés créés avec certains éditeurs d'images, tels que Affinity Designer, cassés.

Dossiers de sprites ⭐
folders

Ajoute des dossiers à la liste des sprites, ainsi qu'aux costumes et aux sons. Pour créer un dossier, faites un clic droit sur n'importe quel sprite et cliquez sur "créer un dossier". Cliquez sur un dossier pour l'ouvrir ou le fermer. Faites un clic droit sur un sprite pour voir dans quels dossiers vous pouvez le déplacer, ou alors glissez le dans un dossier ouvert.

Bouton copier le code sur les forums
forum-copy-code

Ajoute un bouton « copier le code » au-dessus des boîtes de code dans les messages du forum qui copie le contenu dans le presse-papiers.

Citer le numéro de poste
forum-id

Cliquez sur le bouton "Citer le numéro du post" pour inclure un lien vers le post dans votre réponse.

Citations et blocs de code personnalisables sur les forums
forum-quote-code-beautifier

Affiche du code et des citations de manière plus agréable sur les forums.

Recherche sur les forums
forum-search

Ajoute une barre de recherche de posts sur la page d'accueil des forums. Utilise ScratchDB pour obtenir les informations.

Utiliser le fuseau horaire local dans les forums
forum-time-zones

Utilise votre fuseau horaire local pour afficher les dates et heures de publication sur les forums.

Bouton Plus de forums dans la barre d'outils
forum-toolbar

Ajoute des boutons dans la barre d'édition du forum qui ajoute des tags BBCode, comme [center] ou [color].

Zones complètes ⭐
full-signature

Affiche le contenu complet des zones à hauteur limitée, telles que les signatures d'utilisateurs sur les forums et Ce que j'ai fait sur les profils, en les rendant redimensionnables ou défilables.

Plein écran amélioré
fullscreen

Corrige certains effets indésirables dans le mode plein écran des projets, l'ouvre dans le mode plein écran de votre navigateur et masque la barre d'outils du drapeau vert.

Support manette
gamepad

Interagissez avec les projets en utilisant une manette USB ou Bluetooth.

Miniatures haute résolution
hi-res-thumbnails

Remplace les miniatures du site Web par des images de plus haute résolution. Utile pour les écrans Retina (haute densité de pixels) et la navigation avec un zoom supérieur à 100 %.

Cacher le bouton supprimer
hide-delete-button

Masque le bouton de suppression (icône de la corbeille) des sprites, costumes et sons. Ils peuvent toujours être supprimés à l'aide du clic droit.

Masquer automatiquement la palette de blocs ⭐
hide-flyout

Cache la palette des blocs si elle n'est pas survolée. Cliquez sur l'icône de cadenas pour la maintenir en place temporairement. Sinon, utilisez le mode "Clic sur la catégorie".

Masquer les nouvelles variables
hide-new-variables

N'affiche pas automatiquement les nouvelles variables ou listes.

Cacher les stats du projet
hide-project-stats

Vous permet de masquer les loves, les favoris, les remix et le nombre de vues sur la page d'accueil et les pages de projet.

Cacher la signature sur les forums
hide-signatures

Masque les signatures dans les messages des forums et ajoute un bouton "Afficher la signature".

Uploader des images sur les forums
image-uploader

Ajoute un bouton pour télécharger et insérer directement des images dans les messages du forum.

Scrolling infini
infinite-scroll

Permet un scrolling sur différents endroits du site sans avoir à cliquer sur "montrer plus".

Position d'un nouveau sprite personnalisable
initialise-sprite-position

Changer les position par défaut des ordonnées et abscisses lors de la création d'un nouveau costume

Plus d'articles par ligne
items-per-row

Vous permet de modifier le nombre d'éléments affichés sur une seule colonne dans une grille de projets, de studios ou d'utilisateurs.

Afficher les dates de modification et du partage du projet
last-edit-tooltip

Passez la souris sur la date de partage d'un projet pour savoir quand il a été modifié pour la dernière fois.

Projet à l'affiche jouable
live-featured-project

Affiche une bannière avec le projet a l'affiche de l'utilisateur directement sur son profil. Basé sur l'implémentation de Mega Scratch Userscript.

Ajout automatique d'extensions
load-extensions

Ouvre automatiquement la musique, le stylet et d'autres extensions dans le menu des catégories de blocs de l'éditeur.

Plus grande limite de caractères dans la section "Ce que je suis en train de faire"
longer-wiwo

Vous permet de taper 55 caractères de plus dans la section "Ce que je suis en train de faire" sur votre profil. N'importe qui sur Scratch verra les caractères en plus.

Meilleurs boutons pour l'éditeur de commentaires du forum
material-forum-editor-buttons

Modifie les icônes des boutons dans l'éditeur de messages de forum de bitmaps pixélisés en icônes de matériaux.

Enregistreur vidéo de projet ⭐
mediarecorder

Ajoute un bouton "démarrer l'enregistrement" à la barre de menu de l'éditeur qui vous permet d'enregistrer la scène du projet.

Filtres de messages
message-filters

Vous permet de choisir quels types de messages montrer ou cacher sur la page des messages de Scratch

Plus de liens
more-links

Affiche des liens pour les URLs provenant d'ailleurs que scratch.mit.edu.

Position de la souris ⭐
mouse-pos

Affiche la position X/Y de la souris a côté du drapeau vert dans l'éditeur.

Déplacer le costume en haut ou en bas
move-to-top-bottom

Ajoute un élément au menu contextuel du clics droit pour déplacé un costume ou un son en haut ou en bas de ça liste. Précédemment dans le Scratch Developer Tools.

Nombre de messages sur l'icône de l'extension ⭐
msg-count-badge

Affiche le nombre de messages non lus sur l'icône de Scratch Addons

Mode lecture de projet en sourdine ⭐
mute-project

Ctrl + Clic sur le drapeau vert pour couper/rétablir le son du projet.

Intégration d'ocular
my-ocular

Affiche les statuts ocular et les réactions aux messages du forum. Utilise my-ocular.jeffalo.net pour obtenir des données.

Nommer les scripts avant de les mettre dans le sac-à-dos
name-backpack-items

Vous demande de nommer les scripts après les avoir fait glisser dans le sac à dos, au lieu d'utiliser le nom par défaut "code".

Ne pas espacer automatiquement les scripts qui se chevauchent
no-script-bumping

Permet aux scripts d'être déplacés et modifiés sans provoquer le déplacement des scripts qui se chevauchent.

Ancienne disposition des studios
old-studio-layout

Rétablit la disposition de la page des studios à celle utilisée avant la mise à jour des studios ou à celle qui était précédemment disponible grâce à l'addon Scratch 2.0 → 3.0.

Pelure d'oignon ⭐
onion-skinning

Affiche des superpositions transparentes des costumes précédents ou suivants lors de l'édition d'un costume. Contrôlé par des boutons sous l'éditeur de costumes par les boutons de zoom.

Éditeur de costume par défaut ⭐
paint-by-default

Modifie l'action par défaut des boutons "Choisir un Sprite/Costume/Backdrop/Son", qui ouvrent la bibliothèque par défaut.

Bouton pause ⭐
pause

Ajoute un bouton pause a côté du drapeau vert.

Barre de chargement du projet ⭐
progress-bar

Ajoute une barre de chargement lors du chargement et la sauvegarde des projets.

Nombre de sprites et de scripts
project-info

Affiche le nombre de sprites et de scripts que contient un projet.

Onglets pour les notes du projet
project-notes-tabs

Crée des onglets sur les pages de projets pour les sections "Instructions" et "Notes et crédits" du projet.

Redirection vers les forums standards
redirect-mobile-forums

Vous redirige vers les forums standard ("principaux") lorsque vous cliquez sur un lien vers les forums mobiles.

Bouton Remix dans vos propres projets
remix-button

Ajouté le Bouton de remix à vos propres projets.

Bouton d'arborescence des remix ⭐
remix-tree-button

Ramène le bouton de l'arbre des remix sur les projets, près du bouton "Copier le lien"

Enlever les bords arrondis de la scène
remove-curved-stage-border

Enlève les bords arrondis de la scène, vous permettant de voir les coins.

Confirmation pour la suppression de sprite ⭐
remove-sprite-confirm

Vous demande si vous êtes sûrs de supprimer un sprite dans un projet.

Formulaire de commentaire redimensionnable ⭐
resizable-comment-input

Fait en sorte que la zone d'écriture de commentaire puisse être redimensionnable dans les pages en style Scratch 3.0.

Messagerie Scratch ⭐
scratch-messaging

Disponible lorsque vous cliquez sur l’icone de Scratch Addons. Permet une lecture et un moyen pour répondre simple pour vos messages Scratch : groupés par projets, montre les commentaires entiers avec leur contexte, donne un moyen de répondre direct.

Notifications Scratch ⭐
scratch-notifier

Notification lors de nouveaux messages Scratch. Cliquez sur la notification pour aller directement sur le message (commentaire, projet, etc.)

Scratchblocks 3.0 sur les forums
scratchblocks

Active la nouvelle version des blocs Scratch sur les forums.

Scratch 2.0 → 3.0 ⭐
scratchr2

Fait en sorte que les pages de Scratch 2.0 ressemblent à celles de Scratch 3.0.

Statistiques du profil
scratchstats

Affiche les statistiques provenant de ScratchDB sur les pages de profil.

Aligner les scripts sur la grille
script-snap

Les script vont automatiquement s'aligner sur les points de l'arrière-plan.

Barre de recherche sur dans Mes projets
search-my-stuff

Vous autorise à recherchez vos projets, studios et et projet supprimé dans la page Mes projets. Commencez simplement à écrire n'importe où pour commencé la recherche. Vous pouvez aussi faire Ctrl + Entrée pour ouvrir rapidement le premier projet trouver.

Rechercher des profils ⭐
search-profile

Vous permet d'aller sur le profil d'un utilisateur lorsque vous recherchez son pseudo.

Recherche de sprites par nom
search-sprites

Ajoute un champ de recherche au tableau des sprites pour rechercher des sprites par nom.

Bug du point-virgule 🥚
semicolon

Ramène le bug du point-virgule sur Scratch.

Bouton source BBCode
show-bbcode

Ajoute un bouton sur les forums qui affiche la source BBCode d'un post.

Bouton Parcourir les abonnés dans les studios
studio-followers

Ajoute un bouton en haut de l'onglet « Curateurs» dans les studios que vous gérez, qui répertorie vos abonnés et utilisateurs suivis et vous permet d'inviter un utilisateur affiché en un seul clic.

Outils de studio ⭐
studio-tools

Permet aux managers de promouvoir facilement les curateurs au rang de managers ou de retirer n'importe qui d'un studio dans l'onglet "curateurs", et ajoute la possibilité de quitter un studio à l'aide d'un bouton en-dessous de "Suivre le studio".

Basculer les variables entre "Pour tous les sprites" et "Pour ce sprite uniquement.
swap-local-global

Ajoute plus d'options lors du renommage d'une variable ou d'une liste existante : permet de basculer entre « Pour tous les sprites » et « Pour ce sprite uniquement » et si les variables sont stockées dans le cloud. Ajoute également une nouvelle option lors d'un clic droit sur une variable/liste pour changer rapidement sa portée.

Transparence des blocs
transparent-orphans

Ajuste la transparence des blocs dans l'éditeur, avec des options séparées pour les blocs orphelins (ceux sans chapeau en haut de la pile) et les blocs qui sont déplacés.

Vrais liens YouTube dans les forums
true-youtube-links

Remplace les liens vers le site intégré par de vrais liens YouTube.

Bouton TurboWarp
turbowarp-player

Ajoute un bouton sur les pages de projet pour lire le projet dans TurboWarp.org, à côté du bouton Remix.

Masquer le bouton Tutoriels
tutorials-button

Masque le bouton Tutoriels dans la barre de menus de l'éditeur.

Afficher l'ID des utilisateurs
user-id

Ajoute l'ID utilisateur à côté du nom d'utilisateur en haut des pages de profil.

Gestionnaire de variables
variable-manager

Ajoute un nouvel onglet à côté de "sons" dans l'éditeur pour mettre à jour facilement les variables et les listes.

Plein écran pour YouTube
youtube-fullscreen

Affiche le bouton plein écran dans le lecteur Youtube de Scratch.
